# API de Usuários - Spring Boot + MySQL

Este projeto é uma API REST desenvolvida em **Java Spring Boot** com integração ao **MySQL**.  
A aplicação implementa um CRUD completo de usuários, com **mapeamento JPA**..

---

## Funcionalidades
- Criar usuário (POST)
- Listar todos os usuários (GET)
- Buscar usuário por ID (GET)
- Atualizar usuário (PUT)
- Deletar usuário (DELETE)

---

## Tecnologias Utilizadas
- **Java 17+**
- **Spring Boot 3+**
- **Spring Data JPA**
- **MySQL**
